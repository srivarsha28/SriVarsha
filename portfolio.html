Stock Volatility Prediction Using Public Sentiment and Financial Data

Developed a machine learning model to analyze stock price movements based on public sentiment from forums like Reddit and Yahoo Finance, along with financial performance metrics. Leveraged RoBERTa and FinBERT for NLP-based sentiment analysis, automated data extraction with BeautifulSoup and Selenium, and trained models using LightGBM and Scikit-Learn. The project helped identify undervalued and overhyped stocks for better trading strategies.

U-Net Model for Lung and Infection Segmentation

Built a U-Net-based deep learning model for lung and infection segmentation in chest CT scans, aiding in the diagnosis of diseases such as COVID-19. Preprocessed medical images using CLAHE, cropping, resizing, and data augmentation. Implemented a dual U-Net framework to enhance segmentation accuracy and designed a real-time interface for testing on new patient scans.

Chronic Diseases Prediction Using Machine Learning

Developed a machine learning model to predict diseases like diabetes, heart disease, brain tumors, and lung cancer. Used structured datasets from Kaggle and UCI, applied feature engineering, and optimized prediction accuracy using ensemble learning methods. Integrated a Flask-based web application for real-time user input and disease risk assessment.
